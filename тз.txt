Программа - это 2д игра с видом сверху.
Персонаж - квадрат, также как и враги.
игра происходит, по сути, на клеточном поле.
Есть 5 уровней.
Карта и картинки для каждого уровня задаются в отдельной папке.
Каждый уровень отличается, и сложнее предыдущего.
Чтобы пройти уровень, нужно дойти до особой клетки(выход) и простоять на этой клетке 3 секунды.
Стартовое окно - меню с тремя кнопками(играть, статистика, выход).
Нажав кнопку играть, появляется меню  с 5 уровнями и кнопкой "назад".
У героя 3 единицы здоровья, которые отображаются на экране.
При ранении герой некоторое время (+- 1 сек) не может получать урон.
Статистика храниться в json файле
При нажатии escape, уровень ставиться на паузу.
Если пройти все уровни, то появиться финальное, поздравительное окно(только 1 раз)
Все составляющие программы лежат в репозитории github
ссылка на репозиторий https://github.com/GeorgiiChertkoev/pygame-project

Классы: 
Hero - герой
Tile - поле
Enemy - враг
Hearts - сердечки(здоровье)

все классы имеют функцию update, и состоят в группах спрайтов.

есть 3 типа поля:
- пол
- стена
- выход

Карта уровня - txt файл

Используються библиотеки(минимум) pygame, sys, json

По возможности: 
- звуки и музыка для уровней и главного меню
